# AOEE - Streamlit Cloud Deployment

This app is deployed on Streamlit Cloud and uses a sample dataset (5,000 records) for demonstration purposes.

## Live Demo
The full dataset contains 228,572 records. This demo uses a representative sample to ensure fast loading times.

## Features
- What-If Simulator: Test mobile van deployment scenarios
- ROI Predictor: Calculate intervention cost-effectiveness
- Auth Success Intelligence: Identify hardware failure hotspots
